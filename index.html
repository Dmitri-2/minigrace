Minigrace

Minigrace is a compiler for the Grace programming language.

Minigrace compiles Grace to native code via C, and to JavaScript. The compiler is written in Grace itself. It supports most of the specified language to date: for details on current language feature support see .

Minigrace should work on POSIX-compliant systems, and is known to work on Linux, NetBSD, and Mac OS X. Native code generation requires a properly-configured GCC on the path, which may require installing development tools. Running the generated JavaScript requires that node.js be installed.

Tarball distributions should compile with just ./configure && make. More detailed build instructions are in .

After building there will be a minigrace executable in the root directory. This executable supports a few different modes and options, described in . The simplest usage is just: ./minigrace file.grace which will compile file.grace to a native executable and then run it, and ./minigrace --target js file.grace which will compile file.grace to JavaScript and then execute it with node.
